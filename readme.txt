Fetch content from any website

== Description ==

This plugin gives the user the ability to fetch content from any website and show the data using simple shortcode. Here is a <a href="https://wpsupports.net/content-fetcher/">demo</a> 

Sample shortcode [get_content_from url='https://shaplakanon.com/wpsupport/content-fetcher/' with='.entry-title']

= Features:  =

* simple shortcode


== Installation ==

1. Download the plugin

2. upload to plugin directory

3. activate from dashboard

4. There is no settings

5. Just insert this shortcode to your post,page or in widget [get_content_from url="http://domain.com" with="id_or_class"]
